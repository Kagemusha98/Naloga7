# Projekt: Spletna aplikacija Knjižnica

## Glavne funkcionalnosti

1. **Uporabniška prijava in registracija**  
   Omogoča ustvarjanje računa, prijavo ter ponastavitev gesla.

2. **Nadzorna plošča (Dashboard)**  
   Prikazuje ključne informacije in statistiko za uporabnika.

3. **Urejanje uporabniškega profila**  
   Sprememba osebnih podatkov, slike profila in nastavitev.

4. **Izmenjava podatkov prek API-ja**  
   Aplikacija omogoča pošiljanje in pridobivanje podatkov s strežnika.

5. **Upravljanje vsebine (CRUD)**  
   Dodajanje, urejanje, brisanje in pregledovanje elementov znotraj aplikacije.
